# SiteSprint BR

Automacao de prospeccao e acompanhamento comercial para pequenos negocios locais.

Este repositorio contem:

- busca diaria de leads sem site
- integracao com Airtable
- worker de follow-up comercial
- templates cliente-facing
- workflows de execucao diaria

Regra principal: mensagens para clientes nunca devem mencionar ferramentas internas, automacoes, CRM, APIs ou bastidores tecnicos.
